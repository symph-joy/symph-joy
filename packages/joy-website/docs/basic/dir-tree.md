# 应用目录

```yaml
    myapp
        src # 源代码目录
            client  --客户端代码
                pages    # 约定React路由目录
                models    # ReactModel目录
                components  # React组件
            server  --服务端代码
                controllers
                services
        config  # 配置目录
            config.ts
        public  # 静态公共目录
            logo.png

```
